这是什么？

TaiJianKiller 是一个 AI 续写工具。上传你的小说原著，它会先读懂风格和故事，再生成续写章节，还会给你一份质量对比报告。

2 / 3

你需要准备什么？

一个 .txt 格式的小说文本文件就够了。支持 UTF-8、GBK、GB18030 编码，上传后自动处理。其他所有配置都有默认值，不需要手动填写。

没有自己的文本？可以先用内置原创样例试跑，体验完整流程。

3 / 3

怎么开始？

点击左侧表单顶部的「快速开始 · 一键试跑原创样例」按钮，即可体验完整续写流程，不需要上传任何文件。

也可以直接上传你自己的 .txt 文件，然后点「开始分析并续写」。
